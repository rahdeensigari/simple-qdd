# simple-qdd
A small and cheap quasi-direct driven planetary robotic actuator. It uses an 8:1 planetary reduction powered by a 5010 BLDC drone motor recoiled in star configuration to get ~90KV.

<img width="918" height="943" alt="image" src="https://github.com/user-attachments/assets/0f2003c2-ff47-4e0e-a6b5-124573feeb06" />

This is not the first actuator that I have made, and it certainly won't be the last either. The difference between this actuator and the previous ones that I have made is mainly its size and cost. This actuator is much smaller then all the ones that I have made in the past, and because the hardware is literally smaller, it is also much cheaper.
